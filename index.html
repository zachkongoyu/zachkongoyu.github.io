Moss — AI Operating System

rust · tokio · agent runtime

A local-first AI runtime that converts intents into parallel execution plans, generates code on the fly, and coordinates specialist agents through a Blackboard-style shared memory. No fixed tools. No framework lock-in.

The bet: AI doesn't have to think like a human. Give it concurrency, code-gen, and shared state, and it can solve goals that linear "ReAct" agents can't touch.
